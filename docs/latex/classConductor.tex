\doxysubsection{Detailed Description}
Orchestrates execution of source apportionment analysis operations. 

The \doxylink{classConductor}{Conductor} class serves as a command dispatcher and coordinator for various source apportionment algorithms including Genetic Algorithms (GA), Markov Chain Monte Carlo (MCMC), Levenberg-\/\+Marquardt optimization, and statistical analyses. It manages the lifecycle of optimization algorithms, maintains analysis results, and provides an interface between the GUI (\doxylink{classMainWindow}{Main\+Window}) and the computational backend (\doxylink{classSourceSinkData}{Source\+Sink\+Data}).

Key responsibilities\+:
\begin{DoxyItemize}
\item Execute analysis commands with configurable parameters
\item Manage algorithm instances (GA, MCMC)
\item Validate input data for negative concentrations
\item Collect and package analysis results
\item Coordinate with progress reporting UI
\end{DoxyItemize}

\begin{DoxyNote}{Note}
The \doxylink{classConductor}{Conductor} does not own the \doxylink{classSourceSinkData}{Source\+Sink\+Data} -\/ it operates on externally managed data passed via \doxylink{classConductor_a96ce7f8f0838b69e32fbfefef22e4ec5}{Set\+Data()} 
\end{DoxyNote}
