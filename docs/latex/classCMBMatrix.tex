\doxysubsection{Detailed Description}
Matrix class with labeled rows and columns for Chemical Mass Balance analysis. 

Extends CMatrix with string labels for rows and columns, JSON serialization, table widget generation, and specialized operations for CMB calculations. Inherits from both CMatrix (for mathematical operations) and \doxylink{classInterface}{Interface} (for I/O and visualization).
