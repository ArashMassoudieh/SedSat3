\doxysubsection{Detailed Description}
Manages a collection of concentration measurements with statistical analysis. 

Inherits from vector$<$double$>$ to store concentration values and provides\+:
\begin{DoxyItemize}
\item Statistical measures (mean, standard deviation, norms)
\item \doxylink{classDistribution}{Distribution} fitting (normal, lognormal)
\item Box-\/\+Cox transformations
\item Goodness-\/of-\/fit testing (Kolmogorov-\/\+Smirnov) 
\end{DoxyItemize}
