\doxysubsection{Detailed Description}
Collection of time series with labels and observed values. 

Extends Time\+Series\+Set$<$double$>$ with custom labels for time points, observed values for comparison, JSON serialization, and table generation. Used for organizing multiple time series data such as contribution time series from different sources or fitted vs observed distributions.
