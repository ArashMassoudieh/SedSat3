\doxysubsection{Detailed Description}
Vector class with string labels for Chemical Mass Balance analysis. 

Extends CVector with element labels, JSON serialization, table widget generation, and specialized operations for CMB calculations. Supports both numeric and boolean display modes, sorting operations, and range-\/based extraction. \begin{Desc}
