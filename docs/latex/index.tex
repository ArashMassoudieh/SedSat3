\chapter{Sed\+SAT3 (Sediment Source Assessment Tool, version 1.0.25)}
\hypertarget{index}{}\label{index}\index{SedSAT3 (Sediment Source Assessment Tool, version 1.0.25)@{SedSAT3 (Sediment Source Assessment Tool, version 1.0.25)}}
\label{index_md_README}%
\Hypertarget{index_md_README}%
 Sed\+SAT3 is an open-\/source software package developed by the U.\+S. Geological Survey for sediment source fingerprinting using chemical and isotopic tracers. It provides a graphical user interface and a suite of tools for preprocessing, tracer selection, source apportionment, and diagnostic analysis. \hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Cross-\/platform GUI}} built with Qt6 for managing projects, configuring simulations, and visualizing results. ~\newline

\item {\bfseries{Data import}} from Microsoft Excel files with a standardized tab structure. ~\newline

\item {\bfseries{Preprocessing tools}} including outlier detection, particle size and organic matter corrections, bracketing analysis, and discriminant function analysis (DFA). ~\newline

\item {\bfseries{Statistical routines}} such as ANOVA, automated tracer selection, tracer discriminant power evaluation, distribution fitting, correlation matrices, t-\/tests, error analysis, Kolmogorov–\+Smirnov tests, and Box–\+Cox transformations. ~\newline

\item {\bfseries{Source apportionment methods}}\+:
\begin{DoxyItemize}
\item Deterministic maximum likelihood (Levenberg–\+Marquardt optimization) ~\newline

\item Genetic algorithms for global search ~\newline

\item Bayesian inference via MCMC sampling ~\newline

\end{DoxyItemize}
\item {\bfseries{\doxylink{classResults}{Results} output}} with visual and tabular summaries, including credible intervals, posterior distributions, predicted concentrations, and diagnostic plots. ~\newline

\end{DoxyItemize}

\hypertarget{index_autotoc_md18}{}\doxysection{\texorpdfstring{Quickstart Example}{Quickstart Example}}\label{index_autotoc_md18}

\begin{DoxyEnumerate}
\item Prepare your source and target sediment data in Excel format, following the standard tab layout (see sample files in the {\ttfamily examples/} folder). ~\newline

\item Launch Sed\+SAT3 and create a new project. ~\newline

\item Import your Excel file through the GUI. ~\newline

\item Apply preprocessing (e.\+g., tracer selection, corrections). ~\newline

\item Configure an apportionment method (e.\+g., MCMC) and run the analysis. ~\newline

\item View outputs such as posterior distributions, source contribution estimates, and diagnostic plots. ~\newline
