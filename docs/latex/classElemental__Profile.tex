\doxysubsection{Detailed Description}
Container for elemental concentration data of a single sediment sample. 

Stores and manipulates elemental concentration data ("{}fingerprint"{}) for one sediment sample. Inherits from std\+::map$<$string,double$>$ to provide map-\/like access where keys are element names and values are concentrations.

Key capabilities\+:
\begin{DoxyItemize}
\item Storage of element concentrations (ppm, \%, or other units)
\item Quality control marking system for flagging outliers
\item Statistical operations (sorting, ranking, dot products)
\item Correction algorithms (organic matter, particle size effects)
\item Multiple export formats (JSON, table, text)
\item Subsetting and filtering operations
