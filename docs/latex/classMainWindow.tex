\doxysubsection{Detailed Description}
Main application window for Sed\+SAT3 source apportionment analysis. 

Provides the primary user interface for importing data, configuring analyses, executing source apportionment algorithms, and visualizing results. Manages data models, plotting, project file I/O, and coordinates with the \doxylink{classConductor}{Conductor} class for computational operations.
